1.4. Loops#

Loops let you repeat code multiple times without writing it over and over. Use them to process lists of items, count, or repeat an action.

1.4. For Loops#

A for loop repeats code a specific number of times. It tracks a counter variable that changes each iteration.

1.4. While Loops#

A while loop repeats code as long as a condition is true. Use it when you don't know how many times to loop in advance.

1.4. Break and Continue#

  • break stops the loop immediately
  • continue skips to the next iteration

1.4. For...Of Loop#

An easy way to loop through items in an array without managing a counter.
