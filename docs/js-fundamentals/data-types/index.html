Basics ⏱ 1 min read

1.3. Data Types#

JavaScript supports several fundamental data types. Each type tells JavaScript how to interpret and work with the data.

1.3. The Main Data Types#

Numbers: Integer and decimal values
Strings: Text values, enclosed in quotes
Booleans: true or false (for yes/no decisions)
undefined: A variable that hasn't been assigned a value
null: Explicitly means "no value"

1.3. Checking Data Types#

Use the typeof operator to find out what type a value is.

1.3. Type Conversion#

Sometimes JavaScript automatically converts data from one type to another (coercion). Sometimes you need to do it manually.
