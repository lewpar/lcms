Operators

Basics ⏱ 1 min read

1.4. Operators#

Operators are symbols that tell JavaScript to perform an operation on one or more values. Common types include arithmetic, comparison, and logical operators.

1.4. Arithmetic Operators#

Use these to perform math:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
  • ** Exponentiation (power)

1.4. Comparison Operators#

These compare values and return true or false:

  • == Equal to (loose equality)
  • === Equal to (strict equality, recommended)
  • != Not equal to
  • !== Not equal to (strict)
  • > Greater than
  • < Less than
  • >= Greater than or equal
  • <= Less than or equal

1.4. Logical Operators#

These combine boolean values:

  • && AND (true only if both are true)
  • || OR (true if at least one is true)
  • ! NOT (flips true to false and vice versa)

1.4. Assignment Operators#

These assign or update values:

  • = Assign
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
