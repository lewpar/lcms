1.3. Variables#

A variable is a container for storing data values. Think of it as a labeled box where you can put information and retrieve it later.

1.3. Creating Variables#

JavaScript has three ways to create variables: let, const, and var. In modern JavaScript, we use let and const.

1.3. Naming Rules#

Variable names must:

  • Start with a letter, underscore _, or dollar sign $
  • Contain only letters, digits, underscores, and dollar signs
  • Be case-sensitive (name and Name are different)
  • Not be reserved keywords like function, class, etc.

1.3. Practice#

Try creating variables for:

  1. Your favorite color (use const)
  2. Your current mood (use let in case it changes!)
  3. The number of hours you've spent learning JavaScript
