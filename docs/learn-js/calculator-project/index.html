Project: Build a Calculator

Practice & Projects ⏱ 2 min read

1. Practice Project: Build a Calculator#

Let's put everything you've learned together! In this project, you'll build a simple calculator that can add, subtract, multiply, and divide two numbers.

Project Goals

✅ Create variables to store numbers
✅ Use functions to organize your code
✅ Use conditionals to handle different operations
✅ Practice working with parameters and return values

💡 What We're Building

A calculator function that takes three parameters: two numbers and an operation (+, -, *, /), and returns the result.

Challenge Exercises


Try modifying the calculator function for these challenges:


Challenge 1: Add a power operation


Add support for '^' that raises the first number to the power of the second.


Challenge 2: Add a modulus operation


Add support for '%' that returns the remainder.


Challenge 3: Create a history


Create an array that stores previous calculations.


Challenge 4: Smart division


Allow the '/' operation but return the result with only 2 decimal places.




Solution Ideas


For Challenge 1 (Power):


else if (operation === '^') {
  return num1 ** num2;
}



For Challenge 2 (Modulus):


else if (operation === '%') {
  return num1 % num2;
}



For Challenge 3 (History):
Create an array outside the function and push results into it each time the function is called.


For Challenge 4 (Decimal places):
Use Math.round() to round the result:


return Math.round((num1 / num2) * 100) / 100;
