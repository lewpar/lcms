1. For Loops#

A loop repeats code multiple times. A for loop is perfect when you know how many times to repeat.

for (let i = 0; i < 5; i++) {
  // This code repeats 5 times
  console.log(i);
}

The for loop has three parts:

  1. Initialize (let i = 0) - Start counting at 0
  2. Condition (i < 5) - Keep going while i is less than 5
  3. Update (i++) - Add 1 to i each time
