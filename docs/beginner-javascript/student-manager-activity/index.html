1. Activity: Building a Student Manager#

In this activity, you'll practice managing collections of data by building a student manager system. Each exercise below focuses on the same core skill—storing and manipulating information about multiple students—but in different ways.

Learning Goal: Master how to create arrays of objects, loop through them, and manipulate the data.

Start with Exercise 1 below and work your way through all exercises. Each one repeats the same concept in a new context!

1.1. Exercise 1: Create and Display Students#

Start by creating an array of student objects and displaying them. Each student should have: name, grade, and gpa.

Goal: Practice creating arrays of objects and accessing their properties.
