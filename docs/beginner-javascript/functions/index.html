Functions - Reusable Code

Functions & Objects ⏱ 2 min read

1. What are Functions?#

A function is a reusable block of code that performs a specific task. Instead of writing the same code multiple times, you create a function once and call it whenever you need it.

Basic syntax:

function greet() {
  console.log("Hello!");
}

// Call the function
greet();  // Prints: Hello!
greet();  // Prints: Hello!

Functions help you:

  • Avoid repeating code
  • Organize your code better
  • Make code easier to test and maintain

2. Parameters and Arguments#


Functions can accept parameters (input values) to make them more flexible:


function greet(name) {
  console.log("Hello, " + name);
}

greet("Alice");  // Prints: Hello, Alice
greet("Bob");    // Prints: Hello, Bob



You can have multiple parameters:


function add(a, b) {
  console.log(a + b);
}

add(5, 3);  // Prints: 8
add(10, 20); // Prints: 30

3. Return Values#


Functions can return values instead of just printing:


function multiply(a, b) {
  return a * b;
}

let result = multiply(5, 3);
console.log(result);  // Prints: 15



When you use return, the function stops executing and sends back the value:


function checkAge(age) {
  if (age >= 18) {
    return "You are an adult";
  } else {
    return "You are a minor";
  }
}

let message = checkAge(25);
console.log(message);  // Prints: You are an adult
