Loops - Repeating Code

Control Flow ⏱ 2 min read

1. For Loops#

A for loop repeats code a specific number of times:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

This will print: 0, 1, 2, 3, 4

The three parts:

  1. Initialization: let i = 0 - starts the counter
  2. Condition: i < 5 - continues while this is true
  3. Increment: i++ - adds 1 to i after each loop

Think of it like: "starting from 0, while i is less than 5, print i and add 1 to it."

2. Looping Through Arrays#


Loops are perfect for accessing every item in an array:


let fruits = ["apple", "banana", "orange"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}



Or use a simpler for...of loop:


for (let fruit of fruits) {
  console.log(fruit);
}

3. While Loops#


A while loop repeats as long as a condition is true:


while (condition) {
  // Code to repeat
  // Don't forget to change the condition!
}



Example:


let count = 0;
while (count < 5) {
  console.log(count);
  count++;
}



Important: Make sure the condition eventually becomes false, or your loop will run forever!
