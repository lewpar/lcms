Arrays - Working with Lists

Functions & Objects ⏱ 2 min read

1. Creating Arrays#

An array is a list of values stored in a single variable. Arrays use square brackets []:

let fruits = ["apple", "banana", "orange"];
let numbers = [1, 2, 3, 4, 5];
let mixed = ["text", 42, true, null];

Each item in an array has an index (position). Indexing starts at 0:

let fruits = ["apple", "banana", "orange"];
//              0          1           2

console.log(fruits[0]);  // "apple"
console.log(fruits[1]);  // "banana"
console.log(fruits[2]);  // "orange"

2. Array Methods#


Arrays have built-in methods (functions) to manipulate them:






Method
Does what
Example





.push()

.pop()
Remove from end

.shift()
Remove from start
arr.shift()




.unshift()

.length

.indexOf()
Find position

Example:


let items = [1, 2, 3];
items.push(4);        // [1, 2, 3, 4]
items.pop();          // [1, 2, 3]
items.unshift(0);     // [0, 1, 2, 3]

3. Advanced Array Methods#


Powerful methods for working with arrays:


let numbers = [1, 2, 3, 4, 5];

// map() - transform each item
let doubled = numbers.map(num => num * 2);
// [2, 4, 6, 8, 10]

// filter() - keep items that match
let evens = numbers.filter(num => num % 2 === 0);
// [2, 4]

// find() - get first matching item
let firstEven = numbers.find(num => num % 2 === 0);
// 2

// some() - check if ANY item matches
let hasEven = numbers.some(num => num % 2 === 0);
// true
