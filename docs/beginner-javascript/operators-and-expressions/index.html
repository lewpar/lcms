1. Arithmetic Operators#

Arithmetic operators perform mathematical calculations:

Operator Name Example Result
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division 10 / 5 2
% Modulo (remainder) 10 % 3 1
** Exponentiation 2 ** 3 8

You can combine these in expressions: (5 + 3) * 2 = 16

2. Comparison Operators#


Comparison operators compare two values and return true or false:






Operator
Meaning
Example





==
Equal (loose)
5 == "5" returns true




===
Equal (strict)
5 === "5" returns false




!=
Not equal
5 != 3 returns true




!==
Not equal (strict)
5 !== "5" returns true




>
Greater than
10 > 5 returns true




<
Less than
3 < 5 returns true




>=
Greater or equal
5 >= 5 returns true




<=
Less or equal
3 <= 5 returns true





Pro Tip: Always use === and !== instead of == and !=!

3. Logical Operators#


Logical operators combine boolean values:






Operator
Name
Meaning
Example





&&
AND
Both must be true
(5 > 3) && (3 > 1) = true




||
OR
At least one must be true
(5 > 3) || (1 > 3) = true




!
NOT
Reverses true/false
!(5 > 3) = false





These are essential for creating complex conditions in your code.
