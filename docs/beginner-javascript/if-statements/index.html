If Statements & Control Flow

Control Flow ⏱ 2 min read

1. Making Decisions with If#

An if statement lets your code make decisions based on conditions:

if (condition) {
  // This code runs if condition is true
}

Example:

let age = 25;
if (age >= 18) {
  console.log("You are an adult!");
}

The code inside the braces {} only runs if the condition is true.

2. If...Else Statements#


Use else to run code when the condition is false:


if (condition) {
  // Runs if condition is true
} else {
  // Runs if condition is false
}



Example:


let number = 5;
if (number > 10) {
  console.log("Number is greater than 10");
} else {
  console.log("Number is 10 or less");
}

3. If...Else If...Else#


Use else if to check multiple conditions:


if (condition1) {
  // Runs if condition1 is true
} else if (condition2) {
  // Runs if condition2 is true
} else {
  // Runs if all conditions are false
}



Example:


let score = 85;
if (score >= 90) {
  console.log("A - Excellent!");
} else if (score >= 80) {
  console.log("B - Good!");
} else if (score >= 70) {
  console.log("C - Passing");
} else {
  console.log("F - Try again");
}
