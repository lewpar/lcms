Selecting & Manipulating DOM

DOM & Events ⏱ 2 min read

1. What is the DOM?#

The DOM (Document Object Model) is a representation of your HTML page that JavaScript can access and modify.

Think of it like this:

  • HTML = The structure of a web page
  • CSS = The styling
  • JavaScript = Can change the HTML and CSS while the page is running

With JavaScript, you can:

  • Select elements from the page
  • Change their text content
  • Change their styles
  • Add or remove elements
  • Respond to user interactions (clicks, typing, etc.)

JavaScript communicates with the web page through a global object called document.

2. Selecting Elements#

Before you can change an element, you need to select it. Here are the most common methods:

// Select by ID (best for unique elements)
let element = document.getElementById("myId");

// Select by class name (can match multiple)
let elements = document.getElementsByClassName("myClass");

// Select by tag name
let paragraphs = document.getElementsByTagName("p");

// Modern way - select using CSS selectors
let element = document.querySelector("#myId");        // By ID
let elements = document.querySelectorAll(".myClass"); // By class
let element = document.querySelector("p");            // By tag

Pro Tip: querySelector and querySelectorAll are more flexible and recommended for modern JavaScript.

3. Changing Text Content#

Once you've selected an element, you can change what it displays:

let heading = document.querySelector("h1");

// Change the text content
heading.textContent = "Welcome to JavaScript!";

// Or use innerHTML to add HTML tags
heading.innerHTML = "<strong>Welcome!</strong>";

Difference:

  • textContent - treats everything as plain text
  • innerHTML - allows you to include HTML tags

Use textContent unless you specifically need HTML tags.

4. Changing Styles#


You can change an element's CSS styles using the .style property:


let box = document.querySelector(".myBox");

// Change individual styles
box.style.color = "red";
box.style.backgroundColor = "yellow";
box.style.fontSize = "20px";
box.style.borderRadius = "10px";



Important: CSS property names with hyphens become camelCase in JavaScript:


    

  • background-color → backgroundColor
    • 

  • font-size → fontSize
    • 

  • border-radius → borderRadius
    • 



This is because hyphens aren't allowed in JavaScript property names!
