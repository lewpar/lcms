Objects - Structured Data

Functions & Objects ⏱ 2 min read

1. What are Objects?#

An object is a collection of related data and methods organized as key-value pairs:

let person = {
  name: "Alice",
  age: 25,
  city: "New York"
};

Access values using dot notation or brackets:

console.log(person.name);      // "Alice"
console.log(person["age"]);    // 25

// Change values
person.age = 26;
person.city = "Boston";

// Add new properties
person.job = "Developer";

Objects are great for grouping related information about one thing (a person, product, book, etc.)

2. Objects with Methods#


Objects can contain functions, called methods:


let calculator = {
  add: function(a, b) {
    return a + b;
  },
  subtract: function(a, b) {
    return a - b;
  }
};

calculator.add(5, 3);      // 8
calculator.subtract(10, 4); // 6



Or using shorthand (modern JavaScript):


let calculator = {
  add(a, b) {
    return a + b;
  },
  subtract(a, b) {
    return a - b;
  }
};

3. Arrays of Objects#


Often you'll have an array containing multiple objects:


let students = [
  { name: "Alice", age: 20, grade: "A" },
  { name: "Bob", age: 21, grade: "B" },
  { name: "Charlie", age: 20, grade: "A" }
];

// Access items
console.log(students[0].name);  // "Alice"
console.log(students[1].age);   // 21

// Loop through
for (let student of students) {
  console.log(student.name + " has grade " + student.grade);
}



This pattern is very common when working with data!
