Creating & Removing Elements

DOM & Events ⏱ 2 min read

1. Creating New Elements#

Instead of hardcoding HTML, you can create elements with JavaScript:

// Create a new element
let newDiv = document.createElement("div");

// Set its content
newDiv.textContent = "Hello!";

// Set attributes and styles
newDiv.id = "myDiv";
newDiv.className = "container";
newDiv.style.color = "blue";

// Add it to the page
let container = document.querySelector("#main");
container.appendChild(newDiv);

Common methods:

  • document.createElement(tagName) - Create a new element
  • element.appendChild(child) - Add element as last child
  • element.insertBefore(newChild, referenceChild) - Insert before a specific element
  • element.textContent = "text" - Set the text
  • element.className = "class" - Set CSS class

2. Building Dynamic Content#


A common pattern is to create elements from data:


let fruits = ["Apple", "Banana", "Orange"];
let list = document.querySelector("ul");

for (let fruit of fruits) {
  let item = document.createElement("li");
  item.textContent = fruit;
  list.appendChild(item);
}



This is powerful when combined with:


    

  • Arrays of data
    • 

  • User input
    • 

  • API responses
    • 



You can build entire interfaces dynamically!

3. Removing Elements#


You can also remove elements from the page:


let element = document.querySelector("#oldElement");

// Remove it from the DOM
element.remove();

// Or, if you know the parent:
element.parentElement.removeChild(element);



Use this for:


    

  • Deleting items from a list
    • 

  • Clearing old content
    • 

  • Cleaning up when not needed
