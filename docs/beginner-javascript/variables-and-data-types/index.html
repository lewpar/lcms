Variables & Data Types

Fundamentals ⏱ 2 min read

1. What are Variables?#

A variable is a container that stores a value. Think of it like a labeled box where you can put data and retrieve it later.

In JavaScript, you can create variables using three keywords:

  • var - old way (avoid in modern code)
  • let - modern, preferred choice
  • const - for values that don't change

Creating Variables

Here's the basic syntax:

let name = "Alice";
let age = 25;
const country = "USA";

Once you create a variable, you can use it anywhere in your code!

2. Data Types#

JavaScript has several built-in data types:

Type Example Description
String "Hello" or 'World' Text data, enclosed in quotes
Number 42 or 3.14 Integers and decimals
Boolean true or false True/false values
Array [1, 2, 3] List of values
Object {name: "Bob"} Collection of key-value pairs
null null Intentionally empty value
undefined undefined Uninitialized variable

You can check the type of a value using the typeof operator.

3. Working with Strings#


Strings are pieces of text. You can combine them, find their length, and transform them:


let greeting = "Hello";
let name = "World";

// Combine strings (concatenation)
let message = greeting + " " + name;

// String methods
greeting.length;          // 5
greeting.toUpperCase();   // "HELLO"
greeting.toLowerCase();   // "hello"
