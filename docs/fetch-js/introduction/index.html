Introduction

Reading ⏱ 2 min read

Modern web applications rarely work alone. A weather app pulls live forecasts from a remote server. A social feed loads new posts as you scroll. A shopping cart checks current prices before checkout. All of these rely on the same core skill: fetching data from the internet using JavaScript.

In this module you will learn everything you need to go from zero to writing real fetch requests in the browser.

1. What you will learn#

This module is broken into the following topics:

  1. JSON — the data format used by almost every web API
  2. APIs — what they are and how the web uses them
  3. HTTP & Responses — the request/response cycle and status codes
  4. The Fetch API — writing fetch calls with async / await
  5. Handling Errors — catching network failures and bad responses
  6. Working with Arrays — iterating over lists of data from an API
Prerequisites

You should be comfortable with basic JavaScript — variables, functions, and objects — before starting this module.
