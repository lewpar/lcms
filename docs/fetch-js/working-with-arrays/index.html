Working with Arrays

Reading ⏱ 2 min read

Many API endpoints return arrays of objects rather than a single record. For example, fetching a list of products, users, or posts will give you back a JSON array. Once you have parsed the response, you can loop over the array using standard JavaScript methods like forEach or for...of.

1. Fetching a List#

Some APIs wrap their array in an object under a named key (e.g. data.quotes), while others return the array directly at the top level. Always check the API documentation or log the response to see what shape it has.

2. Iterating with forEach#

forEach calls a function once for each item in an array. Inside that function you can access the individual object's properties.

4. Filtering Results#


You can use .filter() to narrow down an array to only the items that match a condition, and .map() to transform each item into a new form.
