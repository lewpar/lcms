What is JSON?

Prerequisites ⏱ 2 min read

JavaScript Object Notation (JSON) is a lightweight data format used to store and exchange information between systems. It represents data as key-value pairs inside curly braces, making it easy for both humans and computers to read and write.

The JSON example above contains an object with two key-value pairs that represent pieces of data. In JSON, a key is the name used to identify a piece of information, and the value is the data associated with that key.

JSON Data Types

In JSON, keys are always text (strings) that identify each piece of data, and values can be of different types depending on the information being stored:

  • Strings (text): Store names, labels, or any descriptive information.
  • Numbers: Represent quantities, measurements, or values used in calculations.
  • Booleans (true or false): Indicate binary conditions such as yes/no or on/off.
  • Objects (nested): Group related data together in a hierarchical structure for better organization.

Nesting Objects

In JSON, objects can be nested inside other objects to represent more complex data. This means that a value for a key can itself be another object enclosed in {} with its own key-value pairs.

Nested objects allow you to organise related information hierarchically, making it easier to group details together in a clear structure.
